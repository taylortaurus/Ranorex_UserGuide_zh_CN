Ranorex Studio中的每个测试都以报告结束。但是，你怎么知道测试运行是否成功？该报告详细介绍了从开始到开始的整个测试运行，即有多少测试用例成功，失败或被阻止; 提出了什么错误和警告; 给定测试容器的迭代等等。例如，你可以自定义报告，更改显示的信息或添加自己的徽标，最后，你还可以导出报告。
